### **Installation**

To get started with this project, clone the repository and install the dependencies:

```bash
https://github.com/Mirza2018/aman-ullah-convention-center.git
cd aman-ullah-convention-center
npm install
```

### **Running the App**
```bash
cd aman-ullah-convention-center
npm run dev
```
